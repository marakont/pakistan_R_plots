# Pakistan R training
## 16th - 20th September 2019

Throughout the workshop, the particular functions and examples used will be shaped by several Polio data sets, provided in advance.

Course content:

* Introduction to R
  + History
  + RStudio GUI
  + Console, scripts
* Basic types, indexing
  + base
  + dates and times
* Read and writing data
  + base
  + Excel, web
* Exploring data
  + summary statistics
  + tables
* Pre-processing
  + base: merge, aggregate
  + tidy data: dplyr
  + regular expression
* Plotting
  + base
  + ggplot2
* Maps
